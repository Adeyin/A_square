Exam Count-down manual
Add a new exam time schedule
  • Type in the Course Code in the left space provided
  • Input the exam time in the format of "YYYY/MM/DD"
  • Click "Add" button
  • The information is then stored and list below

Delete a exam time schedule

  • Type in the Course Code in the left space provided
  • Click "Add" button
  • The information is then deleted and remove from the list below

Reset the exam time schedule

  • Click the "Reset" button
  • All the information is then deleted and remove the whole list below

Notification of exam time

  • Pop up notification will show up according to the exam date
  • Make the user notice the exam time and prepare for the exam
